# K-nearest neighbors
K-nearest neighbours works pretty well as supervised classification on this data set.

e.g. 
```
const emotions = [
  'fear',
  'love',
  'sad',
  'happy',
  'anger',
  'excited',
]
```

```
const notEmotions = [
  'bullring',
  'cement',
  'apple',
  'cat',
  'pluto',
  'australia',
]
```

```
'dog' not an emotion
'jealousy' emotion
```
